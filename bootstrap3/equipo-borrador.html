Herramientas

La herramienta utilizada es Bootstrap

Sublimetext como IDE de desarrollo

GitHub (por consola) para el control de versiones, con el repositorio de código en GitHub

Visual Studio Team Services para el desarrollo y la integración contínua

Hostalia para el alojamiento web. Se ha creado un subdominio de desarrollo en bytedesigning y que se han configurado los ficheros robots.txt y el htacces

Slack usamos slack con github y Visual Studio para que nos notifique cuando se hacen los push y las builds.
